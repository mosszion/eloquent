SVG Coordinate System Explained

Original Example with Coordinates Explained

Circle: cx="50" cy="50" r="40"
• Center point is at (50, 50)
• Extends from (10, 10) to (90, 90)

Rectangle: x="100" y="20" width="80" height="80"
• Top-left corner starts at (100, 20)
• Bottom-right corner ends at (180, 100)

SVG Coordinate System Rules

Key Points:
• Origin (0,0) is at TOP-LEFT corner
• X increases going RIGHT →
• Y increases going DOWN ↓ (opposite of math!)
• All measurements are in pixels by default

Shape-Specific Coordinate Properties

Circle

<circle cx="50" cy="50" r="40" />
• cx, cy: Center coordinates
• r: Radius from center
• Actual bounds: (cx-r, cy-r) to (cx+r, cy+r)

Rectangle

<rect x="100" y="20" width="80" height="80" />
• x, y: Top-left corner coordinates
• width, height: Size from that corner
• Bottom-right corner: (x+width, y+height)

Other Common Shapes

Line: <line x1="0" y1="0" x2="100" y2="50" />
• (x1,y1) = start point, (x2,y2) = end point

Polygon: <polygon points="0,0 50,0 25,50" />
• Each pair of numbers is an (x,y) coordinate

Text: <text x="100" y="50">Hello</text>
• (x,y) = baseline position of the text
